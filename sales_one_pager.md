# AIRE Terminal — Sales One Pager

## What it is
AIRE is an underwriting terminal that converts listings into standardized investment reports.

## Why teams buy it
- Faster screening (paste link → decision)
- Consistent underwriting (templates + one scoring system)
- Shareable outputs (PDFs + exports)
- Automation (API + batch)

## Core features
- A–F grade, score, confidence, flags
- Templates (LTR, STR, BRRRR) + custom
- Batch screener (ranked)
- Watchlists + alerts
- PDF export
- Team workspaces
- API keys + endpoint
